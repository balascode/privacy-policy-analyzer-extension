# Quick Start

## Install Extension

1. Open chrome://extensions.
2. Turn on Developer mode.
3. Click Load unpacked.
4. Choose the extension folder.

## Configure API

Edit src/popup/api-config.js and set hostedApiUrl.

## Run First Analysis

1. Open any policy page.
2. Select text.
3. Right-click and choose Analyze Privacy Policy.
4. Review summary and clauses.

## If Results Do Not Show

- Reload extension.
- Confirm backend health endpoint works.
- Check that API returns required fields.
